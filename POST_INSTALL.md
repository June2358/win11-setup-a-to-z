# ⚡ Power User Tips & Customization

윈도우 사용성을 극대화하는 개인 설정, 최적화, 그리고 유용한 팁 모음입니다.

---

## 🎨 1. UI 및 개인화 (Personalization)

### 1.1. 바탕화면 정리
깔끔한 바탕화면 유지를 위한 설정입니다.
- **바탕화면 우클릭** -> `보기`
    - [x] **큰 아이콘** (시원시원한 보기)
    - [x] **아이콘 자동 정렬** (흐트러짐 방지)

### 1.2. 작업 표시줄 다이어트
불필요한 아이콘을 제거하여 작업 공간을 확보합니다.
- **설정** -> `개인 설정` -> `작업 표시줄`
    - [ ] 검색 (아이콘만 표시 or 숨기기)
    - [ ] 작업 보기 (끔)
    - [ ] 위젯 (끔)
    - [ ] 채팅 (끔)

### 1.3. 파일 탐색기 최적화
탐색기를 열었을 때 '즐겨찾기'가 아닌 '내 PC'가 뜨도록 변경하고, 지저분한 기록을 남기지 않습니다.
- **파일 탐색기** -> `...` (더보기) -> `옵션`
    - **파일 탐색기 열기**: [내 PC]로 변경
    - **개인 정보 보호**:
        - [ ] 최근에 사용한 파일 표시 (해제)
        - [ ] 최근에 사용한 폴더 표시 (해제)

### 1.4. 시작 메뉴 클린하게
- **설정** -> `개인 설정` -> `시작`
    - [ ] 최근에 추가된 앱 표시 (해제)
    - [ ] 시작 메뉴, 점프 목록 등에서 최근에 연 항목 표시 (해제)
    - [ ] 검색 기록에서 웹 사이트 표시 (해제)

## ⌨️ 2. 입력 장치 최적화 (Input Devices)

게이밍 및 빠른 작업을 위한 필수 설정입니다.

### 2.1. 키보드 반응 속도
키를 누르고 있을 때 반복 입력되는 딜레이를 최소화합니다.
- **제어판** -> `키보드`
    - **재입력 시간**: [짧게] (최대 우측으로)
    - **반복 속도**: [빠름] (최대 우측으로)

### 2.2. 마우스 가속 끄기
내가 움직인 거리만큼만 정확하게 이동하도록 마우스 가속을 끕니다. (FPS 게임 필수)
- **제어판** -> `마우스` -> `포인터 옵션` 탭
    - [ ] **포인터 정확도 향상** (체크 해제)

## 🚀 3. 시스템 최적화 및 광고 차단 (Optimization & AdBlock)

쾌적한 환경을 위해 광고를 차단하고, 윈도우의 불편한 설정을 수정하는 필수 도구 모음입니다.

### 3.1. 웹 브라우저 광고 차단 (기본)
별도의 프로그램 설치 없이 크롬/엣지 브라우저에서 광고를 완벽하게 차단하고 싶다면 **uBlock Origin**이 정답입니다.
- **[uBlock Origin 다운로드](https://chrome.google.com/webstore/detail/ublock-origin/cjpalhdlnbpafiamejdnhcphjbkeiagm)**
- **특징**: 현존하는 차단기 중 가장 가볍고(CPU/메모리 절약) 성능이 강력합니다.
- *주의: 유사한 이름(uBlock, AdBlock Plus 등)이 아닌 반드시 **'Origin'**이 붙은 것을 설치하세요.*

### 3.2. 시스템 전체 광고 차단 (심화/유료)
웹 브라우저뿐만 아니라 카카오톡 배너, 프로그램 내 광고까지 시스템 레벨에서 차단하려면 **AdGuard** 설치형 프로그램을 추천합니다.
- **[AdGuard for Windows](https://adguard.com/)**: 유료 소프트웨어지만 가장 확실한 솔루션입니다.
- **설정 팁**:
    1. 프로그램 설정에서 **HTTPS 필터링**을 켜야 유튜브 등의 암호화된 광고도 차단됩니다.
    2. 크롬 확장 프로그램은 **[AdGuard Browser Assistant](https://chrome.google.com/webstore/detail/adguard-browser-assistant/bgnkhhnnamicmpeenaelnjfhikgbkllg)**를 설치하세요. (PC 프로그램과 연동됨)
    - *주의: 이 경우 **uBlock Origin을 같이 설치하지 마세요.** (기능 충돌 및 성능 저하 원인)*

### 3.3. 윈도우 커스터마이징 (Winaero Tweaker)
윈도우 설정을 강제로 변경하여 사용성을 개선해 주는 무료 트윅(Tweak) 도구입니다.
- **[다운로드 링크](https://winaero.com/winaero-tweaker/)**
- **추천 설정**:
    - **Shortcut Arrow**: 바탕화면 아이콘의 거슬리는 화살표 제거.
    - **Classic Full Context Menus**: 윈도우 11의 불편한 우클릭 메뉴를 윈도우 10 스타일(한 번에 다 보이기)로 복구.
    - **Disable Windows Update**: (필요시) 윈도우 자동 업데이트 영구 끄기.

## 📂 4. 참고: 한글(HWP) 리소스 경로 정보 (Advanced)

한글 프로그램이 구동될 때 참조하는 주요 이미지 파일의 위치입니다.
시스템 튜닝이나 UI 분석을 원하시는 분들을 위한 참고용 경로입니다.

- **대상 경로**:
  ```
  C:\Program Files (x86)\HNC\Office 2022\HOffice120\Bin\UxXml\Hancom2016\Common\Image\ko-kr\
  ```
  *(※ 주의: 설치된 한글 버전에 따라 `Office 2022`, `Hancom2016` 등의 폴더명은 다를 수 있습니다. 해당 경로가 없다면 상위 폴더에서 유사한 경로를 찾아보세요.)*

- **주요 파일**: **`ci.png`**
  - 해당 파일이 프로그램 실행 시 표시되는 **로고 이미지**입니다.
  - *참고: 이 파일이 존재하지 않거나 로딩되지 않으면, 실행 시 로고가 표시되지 않을 수 있습니다.*
